# Universe Technocite


## How to clone
 1. mkdir ``c:\_\tc``
 2. cd ``c:\_\tc``
 3. git clone https://github.com/thewasmans/u_tc.git
 4. mkdir symlink

Inside symlink, you can place ``Audio``, ``Graphics``, ``Levels`` ... folders
